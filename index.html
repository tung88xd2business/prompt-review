Nhãn viết tắt là cố ý: trần 4.000 ký tự mỗi cảnh, nhãn dài ăn mất chỗ tả hình. Model đọc nhãn ngắn không sao, bảng này là để bạn dò.

Chạy độc lập, không cần claude.ai
  1. Lấy key. Gemini: Google AI Studio → Get API key (có bậc miễn phí). Claude: console.anthropic.com → API keys (trả tiền theo lượt dùng).
  2. Dán key vào mục 12 và chọn đúng nguồn. Key nằm trong trình duyệt máy bạn, không gửi đi đâu ngoài nhà cung cấp đó.
  3. Đừng mở bằng cách bấm đúp file. Mở kiểu file:// hay bị chặn CORS. Mở terminal ở thư mục chứa file rồi chạy python3 -m http.server 8000, vào localhost:8000.
  4. Muốn dùng mọi nơi: kéo file vào Netlify Drop hoặc đẩy lên GitHub Pages. Nhớ là ai mở trang cũng phải tự dán key của họ — key của bạn không nằm trong file.

Trong claude.ai thì để trống mục 12, chạy sẵn không cần key. Mọi thứ còn lại — ảnh, ngân sách ký tự, đếm ký tự, xuất file — chạy y hệt ở cả hai nơi.
